CSS Basics

What is CSS?

CSS stands for Cascading Style Sheets. It is the language used to style the presentation of web pages, including layout, colors, fonts, and overall visual appearance.

Applying CSS to HTML

There are three ways to apply CSS to an HTML document:

  • Inline Styles: Styles applied directly to HTML elements using the style attribute.
  • Internal Styles: Styles defined within a <style> tag in the <head> section of the HTML document.
  • External Styles: Styles defined in a separate CSS file linked to the HTML document.

Basic CSS Syntax


body {
    background-color: lightblue; /* Sets the background color */
}

h1 {
    color: white; /* Sets the text color */
    text-align: center; /* Centers the heading */
}

p {
    font-family: Arial, sans-serif; /* Changes the font of the paragraph */
    font-size: 16px; /* Sets the font size */
}

Why Understanding the Basics of CSS is Valuable

  1. Control Over Appearance:
    CSS allows you to have complete control over the visual presentation of your website. This means you can create unique designs that stand out from the crowd.
  2. Responsive Design:
    With CSS, you can make your website look good on all devices, from desktops to smartphones, using techniques like media queries.
  3. Improved User Experience:
    A well-styled website enhances user experience, making it easier for visitors to navigate and interact with your content.
  4. Separation of Concerns:
    CSS allows you to separate content (HTML) from design (CSS), which makes your code cleaner and easier to maintain.
  5. Career Opportunities:
    Knowing CSS is essential for many web development and design jobs. Employers often look for candidates with strong styling skills.
  6. Frameworks and Tools:
    Many popular frameworks (like Bootstrap) and design tools (like Figma) rely on CSS. Understanding CSS will help you utilize these tools effectively.
