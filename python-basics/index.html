Introduction to Python

Objective

Get familiar with Python syntax and basic concepts.

What is Python?

Python is a high-level programming language known for its simplicity and readability. It’s often referred to as the "English of programming languages" because of its clear syntax, making it easy for beginners to learn and understand.

Uses of Python

  • Web Development: Frameworks like Django and Flask help build dynamic websites.
  • Data Science and Analysis: Libraries such as Pandas and NumPy allow for powerful data manipulation and analysis.
  • Machine Learning: Libraries like TensorFlow and Scikit-learn are used for building machine learning models.
  • Automation and Scripting: Python is great for writing scripts to automate repetitive tasks.
  • Game Development: Libraries such as Pygame can be used to create games.

Why Learn Python?

  1. Simplicity: Python's syntax is straightforward, making it ideal for beginners.
  2. Versatility: Python is used across various domains, from web development to data science.
  3. Strong Community: A large, supportive community means plenty of resources, tutorials, and third-party libraries.
  4. Career Opportunities: Python skills are in high demand in the job market, particularly in tech, finance, and academia.
  5. Open Source: Python is free to use and has a vast number of libraries available for various applications.

Basic Syntax


print("Hello, world!")  # Outputs a message to the console

Variables and Data Types

  • Integers: Whole numbers (e.g., 10).
  • Floats: Decimal numbers (e.g., 3.14).
  • Strings: Text (e.g., "Hello").
  • Lists: A collection of items (e.g., fruits = ["apple", "banana", "cherry"]).

Example

Here’s how you can use variables in Python:


favorite_color = "blue"
age = 25
print("My favorite color is " + favorite_color + " and I am " + str(age) + " years old.")
